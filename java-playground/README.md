# ⚾️ 숫자 야구 기능 목록 Check List

- [ ] 1부터 9까지 서로 다른 랜덤 숫자 3개를 생성한다.
- [ ] 사용자로부터 숫자를 입력 받는다.
    - [예외] 받은 값이 3자리가 아니다.
    - [예외] 받은 값이 숫자가 아니다.
    - [예외] 받은 값에 중복이 있다.
- [ ] 컴퓨터의 수 & 플레이어수를 비교한다.
    - [ ] 몇 개의 숫자가 같은지 확인한다.
    - [ ] 특정 자리에 해당 숫자가 있는지 확인한다.
        - [ ] 같은 숫자만 있으면 `ball`
        - [ ] 같은 숫자 + 같은 위치면 `strike`
        - [ ] 같은 숫자가 하나도 없으면 `낫싱`
        - [ ] 숫자 모두 같은 위치에 있으면 게임 종료.
- [ ] 사용자에게 결과를 알려준다.
- [ ] 재시작 여부를 묻는다.
    -[ ] [예외] 1,2 이외의 값이 포함 되어있다.
